# **App Name**: ClockIT Pro

## Core Features:

- Secure API Key Storage: Allows users to securely store their Clockify API Key and Workspace ID.
- Dynamic Project/Task Selection: Populates project and task dropdowns dynamically via the Clockify API for easy selection.
- Full Time Entry CRUD Operations: Provides complete Create, Read, Update, and Delete functionality for time entries, integrated with the Clockify API.
- Automated Time Entry Suggestions: Generates suggestions for time entry descriptions based on project and task using a generative AI tool.
- Automation Tools: Supports batch creation, predefined templates, and scheduled recurring entries to automate repetitive time management tasks.
- Branded UI: Applies the ClockIT branding with a consistent color theme, updated icons, and the new app name throughout the interface.
- Filtering and Sorting: Enables users to filter time entries by date range, project, task, or billable status for better organization.

## Style Guidelines:

- Primary color: Dark Cyan (#006494) to reflect the original user's brand guidelines. This evokes trust and efficiency.
- Background color: Dark Blue (#13293d) as requested, providing a professional and modern look.
- Accent color: Electric Blue (#7DF9FF) for interactive elements and highlights to provide clarity.
- Body and headline font: 'Inter', a sans-serif font that is great for both headers and body text due to its modern, machined, and objective look.
- Use SVG icons from libraries like Lucide or FontAwesome, customized to match the ClockIT branding, and clearly communicate time tracking functions.
- A clean, intuitive layout that prioritizes usability, with clear sections for time entry, project/task selection, and automation tools.
- Subtle transitions and animations to provide feedback and enhance user experience without being distracting.
